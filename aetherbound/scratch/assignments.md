# Team Assignments

The files in the AetherBound project have been distributed among 8 team members. Each person is assigned files that work together on a specific system component.

## Person 1: Core System & Data Management
**Component:** System Utilities and Data Handling
* `core/logger.py`
* `core/metadata.py`
* `core/settings.py`
* `core/data_manager.py`

## Person 2: User Interaction & Audio
**Component:** Input, Audio, and UI
* `core/audio.py`
* `core/input.py`
* `gameplay/ui.py`

## Person 3: Core Engine & Application Entry
**Component:** Main Application Loop and Setup
* `main.py`
* `run.py`
* `gameplay/engine.py`

## Person 4: Gameplay Mechanics
**Component:** In-Game Tools and Systems
* `gameplay/scanner.py`
* `gameplay/weapons.py`

## Person 5: Physics Simulation
**Component:** Physics Engine and State Management
* `physics/collision.py`
* `physics/engine.py`
* `physics/gravity.py`
* `physics/state.py`

## Person 6: Base Graphics Architecture
**Component:** Graphics Framework and Math
* `graphics/renderer.py`
* `graphics/shaders.py`
* `graphics/camera.py`
* `graphics/frustum.py`
* `graphics/shapes.py`

## Person 7: Specialized Renderers
**Component:** Graphics Rendering Pipeline
* `graphics/renderers/celestial.py`
* `graphics/renderers/effects.py`
* `graphics/renderers/environment.py`
* `graphics/renderers/instanced.py`
* `graphics/renderers/mesh.py`

## Person 8: Asset Loading & 3D Primitives
**Component:** 3D Models and Base Geometry
* `graphics/models/glb_loader.py`
* `graphics/models/mesh.py`
* `graphics/primitives/ring.py`
* `graphics/primitives/sphere.py`
