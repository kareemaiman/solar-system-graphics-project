the goal of this project is to create a 3d space explorer game using opengl and python using the library pyopnegl
the game starts with a spaceship entering earth's solar system the spaceship is viewed from a 3rd person perspective and it is controlled by the mouse like Minecraft player
the game should have 2 separate engines a physics engine and a graphics engine
the physics engine will be focused on simulating Newtonian physics for the game such as gravity, forces, acceleration, velocity, angular momentum and so on any physical calculation will be preformed by the physics engine while any visual calculation will be preformed by the graphics engine the project should have a formal file structure where assets are kept in a directory, the physics engine and all it's dependencies are kept in another directory and the same goes for the graphics engine try to use Data-Oriented Design as much as possible the space ship will be a 3d model imported from an external .glb file with a library called trimesh the spaceship should have 2 main functions first is scanning it scans a planet/ star/moon/asteroid and gives data about it like the most abundant elements in it surface temperature approximate age and so on and the second function is nuclear blasters that shoot at celestial objects and leaves craters on them or if the object is small enough it can completely destroy it
we will use glfw for keyboard and mouse inputs and windowing , numpy and PyGLM for most math
